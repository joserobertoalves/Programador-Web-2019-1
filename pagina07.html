Linguagens para Web

Linguagem HTML
É uma linguagem de marcação de hipertexto.
É utilizada para criar a estrutura de uma página.
Linguagem CSS
É uma linguagem de folhas de estilo em cascata.
É utilizada para formatar uma página web.
Linguagem JavaScript
É uma linguagem de programação voltada para web.
É utilizada para injetar lógica de programação em uma página.
